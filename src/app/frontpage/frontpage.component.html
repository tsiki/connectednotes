CONNECTED NOTES

Connected Notes is a tool for non-hierarchical notes and flashcards. It encourages usage of the Zettelkasten method and spaced repetition, and is open source and free to use.

Connected Notes is in alpha, so expect roughness, bugs and some missing features. Please report all bugs at the project's GitHub page.

Main features

Notes are stored in your personal cloud service, meaning they're accessible from anywhere, including mobile. Currently the only backend option is Google Drive, but Connected Notes is designed with interchangeable backends in mind and I expect more backend options to be implemented.

Connected Notes supports the Zettelkasten method. This means first class support for tagging and connecting notes - in fact, Connected Notes doesn't use folders to organize notes, but uses the tags you've added to structure them (see 'getting started' below).

Notes are written using markdown and stored as plain text markdown files. This means your notes aren't locked in to a proprietary format, and since they're stored on your personal cloud you'll always have access to them, even if this site was to shut down one day.

Connected Notes is free and open source. The project is designed to support notes that will live a long time (as any note taking system should), and open sourcing is an essential part of that promise.

Spaced repetition (ie. flashcards) has been shown to give superior results in learning tasks, not only when measuring the ability to recall, but also when making inferences from learned material (source). The current version of Connected Notes has flashcard support built-in. The support is currently quite simple but more features will appear in the future.

Getting started

Creating a note

Creating a note is as easy as clicking on the 'create note' icon (or pressing ctrl + k) and entering the name of the note. You don't need to choose which folder the note is added to, since folders don't exist - instead, organizing notes is based on tags (see next section). This greatly reduces the friction of creating notes and avoids having to think in strict categories and hierarchies (more).

Tagging notes

Using tags is an integral part of Connected Notes. Since tags essentially replace folders, learning to use them is worthwhile. To add a tag, simply add a Twitter style hashtag anywhere to the note, like #this. You will then see the hashtag appear in the left side menu, and if you click on it, you will see all the notes that contain the tag:

If a note doesn't have any hashtags, it appears under the special 'untagged' tag. Similarly, all notes appear under 'all'. Also, since notes can contain multiple tags, they can also appear under multiple tags:

Tags can be nested under each other, creating a 'tag/subtag' structure. This might be sensible if you have a tag which is clearly a part of some other tag. For example, if you have tags #texas-holdem (a popular card game) and #texas-holdem-strategies, you might not want to have #texas-holdem-strategies appear on the top level.

This can be achieved by dragging the child tag over the parent tag:

Finally, tags can appear under multiple tags. Let's say you have tags #car-brands and #electric-cars. In a traditional folder structure you'd have to make a choice where to place #tesla, but in Connected Notes #tesla could appear under both tags. To achieve this, click on the three dots menu next to #tesla and choose the parent tags.

In comparison, traditional folders are essentially equivalent to each note having a single tag and each tag having at most one other tag as their parent.

Connecting notes

Notes can be connected by typing two square brackets '[[' and selecting the note you want to reference from the autocompletion list. You can also directly write the name of the note [[like so]] and the note will be linked. You can then use ctrl + click to jump to the note. If the note doesn't exist, a new note will be created when jumping into it.

Flashcards

Create a new flashcard by opening a note and pressing ctrl + j. This will open a dialog where you can edit the front and back of the card before saving it (TODO image).

By default, the flashcard is populated with the sentence your cursor is currently on. If you have selected a piece of text before pressing ctrl + j, that text will be prepopulated instead.

To study your flashcards, click on the 'study' icon in the left menu. By default, you'll be shown all due flashcards you have created.

Currently there's no interoperability with Anki but in the future I'd like to have a more Anki-like flashcard experience and allow exporting/importing.

Running your own instance

More details on how to set up your own instance of Connected Notes will be at the GitHub page once the project exits beta.

Caveat

If you're planning to start your own instance, note that Google Drive files are only accessible from whitelisted domains, so notes created in connectednotes.net won't be accessible from someotherdomain.com. The solution is to copy the files and re-upload them via the new correct domain.

Future work

Currently the code base is lacking a lot of tests and has some known issues, but as we finalize the design and structure the number of tests should ramp up and that of issues down.
